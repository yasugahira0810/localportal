localportal
====

localportalは、HTML5のlocalstorageを使ったリンク集作成アプリです。  

## Description

- タグ機能により、ブラウザのブックマーク機能より容易にリンクの管理・検索ができます。
- またInternet Explorerでlocalportalを使うと、ファイルサーバのフォルダをブラウザから開くことができます。
